# PalmArbor Digital Arboretum

A comprehensive, production-ready, multi-page modern web platform for a farmer-focused system that enables small and marginal farmers to confidently adopt oil palm cultivation.

## 🎯 Core Vision
A trust-driven digital ecosystem combining precision advisory, financial clarity, and market linkage for farmers. Built with a desktop-first SaaS approach, featuring rich 3D visualization using React Three Fiber, fluid transitions via Framer Motion, and a cohesive modern UI system powered by Tailwind CSS.

## 🚀 Features

* **Precision Advisory Dashboard:** Interactive 3D plant lifecycle models with soil and weather parameter simulations.
* **Financial Projection Modeler:** 3D animated growth curve simulations matching hectare scope and yield potential to net profitability.
* **Supply Chain Visualizer:** Animated pipeline flow in 3D representing market linkage from local producers to global hubs.
* **Global Community Network:** Abstract 3D particle connectivity map supporting masonry story grids.
* **Sustainability Split-Viewer:** Interactive environment slider combining rendered deforestation data and dense canopy agriculture.
* **Intelligent Nav System:** Modern responsive navigation and multi-layer sidebar with customized routing states.

## 🛠 Tech Stack
* **Framework:** Next.js (App Router, React 19)
* **3D Rendering:** Three.js + React Three Fiber + Drei
* **Styling:** Tailwind CSS + custom glassmorphism architecture 
* **Animations:** Framer Motion
* **Iconography:** Lucide-React

## 📦 Getting Started

1. **Install Dependencies**
```bash
npm install
```

2. **Run Development Server**
```bash
npm run dev
```

3. **Build for Production**
```bash
npm run build
npm start
```

## 🎨 Design System

Our platform incorporates modern UX tenets:
- **Clean Typography:** Calibrated scale for deep data density
- **Subtle Glassmorphism:** Immersive blurs linking the DOM Layer with the WebGL Canvas
- **Immersive Palettes:** Environment-inspired deep greens matching a sleek dark mode.
